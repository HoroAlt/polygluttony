## PUNCTUATION RULES

**IMPORTANT**: Read the ENTIRE conversation context before deciding punctuation. Do NOT default to commas.

### When to use each punctuation mark:

1. **Period (.)** - Complete sentences and statements
   - "I'll handle this."
   - "The technique failed."

2. **Question mark (?)** - ALL questions, always
   - "What do you mean?"
   - "Are you sure about this?"
   - NEVER end a question with a comma or period

3. **Exclamation mark (!)** - Strong emotion, commands, exclamations
   - "Stop right there!"
   - "How dare you!"

4. **Comma (,)** - ONLY when a sentence genuinely continues to the next line
   - Use sparingly - most lines are complete thoughts
   - If using comma, next line MUST start lowercase and complete the sentence
   - When uncertain, prefer period over comma

5. **Ellipsis (...)** - Trailing thoughts, pauses, or interrupted speech
   - "I thought you were..."
   - "But if that's true..."
   - "Wait, unless..."
   - Use when speaker trails off, is interrupted, or leaves thought incomplete
   - Do NOT use ellipsis just because you're unsure about punctuation

6. **NO PUNCTUATION** - Single isolated phrases, names, titles
   - Attack/technique names: "Flame Devouring Heaven Technique"
   - Formation names: "Nine Palace Array"
   - Location labels: "Misty Cloud Sect"
   - Character introductions: "Xiao Yan"
   - These are titles/labels, NOT sentences

### Decision Priority:
1. Is it a question? → Question mark (?)
2. Is it an isolated name/phrase/title? → No punctuation
3. Does the speaker trail off or get interrupted? → Ellipsis (...)
4. Is it a complete statement? → Period (.)
5. Does it genuinely flow into the next line? → Comma (,) + lowercase next

**AVOID**: Defaulting to commas when uncertain. When in doubt, use a period or no punctuation.

## COMMON PATTERNS

- **Sound effects / reactions**: Translate to English equivalents (嗯→Hmm, 哈哈→Haha, 哼→Hmph, 啊→Ah, 呵→Heh, 嘶→Hiss, 喂→Hey)
- **Honorifics**: Translate functionally (前辈→Senior, 师父→Master, 大人→Lord/Sir, 师兄/师姐→Senior Brother/Sister, 师弟/师妹→Junior Brother/Sister)
- **Numbers**: Use digits for large numbers (一百→100, 一万→10,000), words for small ones in dialogue (三个→three, 两人→two people)
- **Cultivation incantations / chants**: Translate for meaning rather than transliterating, preserving the dramatic tone
